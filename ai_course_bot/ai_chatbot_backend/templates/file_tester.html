API Documentation

GET /v1/local-files

Lists files in a directory. Use the directory query parameter to specify a directory.

GET /v1/local-files/hierarchy

Gets a hierarchical tree structure of files and directories. Use directory and max_depth query parameters to customize.

GET /v1/local-files/{file_path}

Retrieves a file. The file_path should be URL-encoded.

GET /v1/local-files/categories

Lists available file categories.

GET /v1/local-files/folders

Lists available file folders.
